# GPT-2 from Scratch — WikiText-2로 학습하기

PyTorch만 사용해서 GPT-2 아키텍처를 처음부터(from scratch) 구현하고, WikiText-2(raw) 데이터셋으로 직접 학습시킨 프로젝트입니다.

Colab 무료 GPU(T4)에서 바로 돌릴 수 있도록 모델 크기와 학습 설정을 맞췄습니다.
